# SDKWork LLM Backend API SDK

This is the SDK family root for the `sdkwork-llm.backend` OpenAPI authority.

- SDK family: `sdkwork-llm-backend-sdk`
- API authority: `sdkwork-llm.backend`
- API prefix: `/backend/v3/api`
- Owner: `sdkwork-llm`
- Standard profile: `sdkwork-v3`
- Authority OpenAPI: `openapi/llm-backend-api.openapi.json`
- Declared generated client: `SdkworkLlmBackendClient`

Generated transport output, when materialized, belongs under:

- `sdkwork-llm-backend-sdk-typescript/generated/server-openapi`

Generated files must not be hand-edited. Fix OpenAPI, route manifests, generator input, or approved composed facades, then regenerate with the canonical SDKWork generator.

Credential mode:

- Open API SDKs use an API key credential provider for protected operations.
- App and backend SDKs use SDKWork dual-token credential injection.
